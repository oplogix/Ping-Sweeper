# Ping-Sweeper
Ping Sweeper written from TCM Securities Practical Ethical Hacking


## Install
* run:  sudo git clone https://github.com/oplogix/Ping-Sweeper.git
* cd Ping-Sweeper/
* .\ipsweep.sh 192.168.1

Will run 192.168.1.1-254
